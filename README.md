# Get-the-News-Articles

## You can do these things:

1. Get new news articles
2. Save articles
3. Add notes to individual articals
4. Remove Notes
5. Remove articles from saved

## Technologies and Libraries Used

- Javascript
- Express JS
- Node JS
- Handlebars
- MongoDB
- Mongoose
- Axios
- Cheerios
- Css
- Bootstrap 4
